##  ASSAMESE WORD OF THE DAY

This is a simple website project named "ASSAMESE WORD OF THE DAY".

The purpose of this project is to show one Assamese word daily along with its English meaning.  
The word changes automatically based on the current date.  
There is also a **Next Word** button to view more Assamese words.

# TECHNOLOGIES USED
- HTML
- CSS
- JavaScript

# FEATURES
- Shows a daily Assamese word
- English meaning is displayed using "=" sign
- Button to view next word
- Simple and clean design

# HOW IT WORKS
JavaScript array is used to store Assamese words.  
The system date is used to select a word for each day.  
CSS is used for styling the page.

This project is made for learning basic web development concepts.
